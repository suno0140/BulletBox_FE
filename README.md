## 1. [Bullet Box] 프로젝트 소개
- 불렛저널 방법을 적용한 일정관리 & 다이어리 서비스
- 다양한 불렛(기호)을 사용한 의미부여 투두리스트
- 자주 쓰는 할 일은 루틴으로, 할 일의 테마는 카테고리로 편리한 사용
